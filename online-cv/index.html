A letter to my guests

Hi,
      Welcome to my very first attempt at a live webpage. Yes, that’s right, I’ve been using the Internet for about 17 years and I have finally decided to join the ranks of the content producers. Why you ask? I’ll tell you... whether you asked or not. It wasn’t until recently that I discovered how much fun programming is. I’ve been studying computers, programming (web and otherwise), and networks for the past 2 years. Programming is what really captured my interest, so here I am putting my work out in the real world.
      You may notice that I haven’t used any images for the paper effect background. (If you’re into that kind of thing) Alas, the code for the background effect is not my own. I used this article’s code and refined it for my purposes. If you’re using an older browser, you’re probably looking at a jumbled page with no paper effect. In that case, I suggest getting the newest version of your browser software. Or better yet, get a browser that updates itself (cough... cough... Chrome). The newest browser software gets more security updates and not only that, it gives web programmers more interesting toys to play with. Everyone loves toys.
      This current version is a meager attempt at expanding my programming knowledge, with little emphasis on the content. If/when I update this site, I intended to play around and have fun on the edge, so don't expect it to work on older browsers. That is all.

“Look on my works, ye Mighty, and despair!” -Percy Bysshe Shelley
